# Vedkamal Portfolio Website

A modern, responsive portfolio website for Vedkamal, a Full Stack Developer. This website showcases skills, projects, and professional experience with a sleek, dark-themed design inspired by cryptocurrency exchange platforms.

## Features

- Modern, responsive design with dark theme
- Animated UI elements using Framer Motion
- Built with Next.js and TypeScript
- Styled with Tailwind CSS
- Optimized for performance and SEO
- Contact form with validation

## Pages

- Home - Introduction and overview
- About - Personal information and experience
- Projects - Showcase of development work
- Skills - Technical abilities and expertise
- Contact - Get in touch form

## Technologies Used

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/vedkamal-portfolio.git
cd vedkamal-portfolio
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Customization

- Update personal information in the component files
- Replace placeholder images with your own in the `/public/images` directory
- Modify color scheme in `tailwind.config.js`

## Deployment

This website can be easily deployed to Vercel, Netlify, or any other hosting platform that supports Next.js.

```bash
npm run build
# or
yarn build
```
